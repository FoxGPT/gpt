Getting Started

Get a key in our discord server, and then:

In your script:

Change... To...
https://api.openai.com/... https://api.hypere.app/...

In Python:

Change... To...
openai.api_key = 'sk-...' openai.api_key = 'key' #get a free key in our discord server!

After that, add the line openai.api_base = 'https://api.hypere.app'

And you should be good to go!

Frequently Asked Questions

Features

Do you support/will you add [any feature]?

Just try it out and see if it works! If you want to suggest something, please contact me.

I get [any error]! How do I fix it?

If you get an error, please contact me. Make sure you copied the code from the example and followed all steps.

Where can I find the documentation?

There is no documentation for this service, because it's based on OpenAI's API.
You can learn more about how to use OpenAI's API here.

General

How is this free?

I'm paying for the server out of my own pocket. If you would like to support this project, please check out the donate section.

Do you collect my ChatGPT/OpenAI session token/API key?

No. It's not even possible for me to do that, unless you leave your OpenAI key in openai.api_key or send it otherwise in the API request.

Does OpenAI know my IP?

No. The user parameter is completely randomized by default.

Do I need to provide any API key?

No. In the official openai Python library, set api_key to ''. In the other libraries, just don't set it at all.
